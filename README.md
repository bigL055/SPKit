## 简介

提供一套基于UIKit的扩展框架,方便于项目快速开发.

## 安装

- 使用cocoapods安装

  ```
  pod 'SPKit'
  ```

- 导入

  - swift

    ```
    import SPKit
    ```

  - objc

    ```
    #import<SPKit/SPKit.h>
    ```
